Smart Crop Recommendation and Monitoring System

Agriculture remains the backbone of many economies, yet farmers often struggle to select the most suitable crops for their soil and season, choose the correct agro-chemicals, and monitor crop health effectively. This project aims to leverage machine learning (ML) and IoT (Internet of Things) to build a system that recommends suitable plants, provides agro-chemical guidance, tracks plant growth, and suggests remedial measures to optimize yield.

Technologies: Programming: Python, JavaScript, ML Frameworks: TensorFlow, Scikit-learn, PyTorch, Chatbot Framework: Dialogflow, Rasa

Expected Outcomes:

  • Functional system ecommending best crop for a given soil and season.
  • Chatbot capable of answering farmers’ queries about crops and agro-chemicals
  • Real-time plant health monitoring and actionable suggestions.
